# QNTMEX Wallet 👑

React Native crypto wallet built with Expo. Royal gold-on-black aesthetic.

## Features
- 5 Screens: Home, Swap, Send, Receive, Settings
- Dark & Light Mode
- Portfolio Dashboard with sparklines
- Token Swap with live rate calculation
- Send Flow with 2-step confirmation
- Receive: SVG QR code with crown mark

## Stack
- Expo SDK 51 / React Native 0.74.5
- React Navigation (Bottom Tabs)
- react-native-svg, expo-linear-gradient
- React Context for theme

## Run
```bash
npm install
npx expo start
```
Scan QR with Expo Go app.
